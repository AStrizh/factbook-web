The World Fact Book

The goal of this project is to allow CIA Factbook users to study the factbook data in a more comprehensive manner.
Note #1: Not all attributes are available. We had to decide which were most relevant for research and discovery.
Note #2: All information is from the 2017 CIA Factbook.

Country Select

Country select lets you look at Factbook data for a specific country, as you would from the official website.


Country Compare

Country compare lets you select several countries and view their data back to back.

Select Countries for comparison


Query Builder

The query builder lets you select which attributes you are interested in and filter on specific criteria.
